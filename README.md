# shoe_website 
📌 Project Overview

This project is a responsive frontend web application developed using HTML, CSS, and JavaScript. It was built to demonstrate core web development skills including UI design, styling, and client-side interactivity.

The project showcases structured layout design, responsive styling, and dynamic behavior using vanilla JavaScript.


🚀 Technologies Used
HTML5 – Structure and content
CSS3 – Styling and responsive design
JavaScript (ES6) – Interactive functionality
Font Awesome – Icons
Custom Fonts & Assets


✨ Features
Responsive layout design
Clean and structured UI
Interactive elements using JavaScript
Integrated icons using Font Awesome
Organized project structure


📂 Project Structure
├── index.html
├── style.css
├── script.js
├── Fonts/
├── font-awesome/
├── img/
└── README.md

🎯 Purpose

This project was developed as a practice-based learning project to strengthen frontend development skills and gain hands-on experience in building user-friendly web interfaces.

📖 How to Run
Download or clone the repository
Open index.html in any modern web browser
